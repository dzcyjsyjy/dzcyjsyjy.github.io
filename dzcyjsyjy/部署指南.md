## 🚀 部署步骤

### 第一步：购买域名和服务器

1. 注册阿里云账号：https://www.aliyun.com
2. 购买域名（如 dzcyjsyjy.com）
3. 购买ECS服务器（推荐选择：CentOS 7.9 或 Ubuntu 20.04）
4. 完成实名认证

### 第二步：域名备案（重要）

**备案流程：**
1. 登录阿里云备案系统
2. 填写网站信息
3. 上传资料（营业执照、法人身份证等）
4. 等待审核（约10-20个工作日）

**备案所需材料：**
- 营业执照
- 法人身份证
- 网站负责人身份证
- 域名证书
- 网站备案核验单

### 第三步：服务器环境配置

#### 1. 连接服务器
```bash
ssh root@your_server_ip
```

#### 2. 安装Nginx
```bash
# CentOS
yum install -y nginx

# Ubuntu
apt update
apt install -y nginx
```

#### 3. 启动Nginx
```bash
systemctl start nginx
systemctl enable nginx
```

### 第四步：上传网站文件

#### 方法一：使用SCP上传
```bash
# 在本地电脑执行
scp -r D:\达州产业技术研究院官方网站\dzcyjsyjy\* root@your_server_ip:/var/www/html/
```

#### 方法二：使用FTP工具
- 推荐工具：FileZilla
- 服务器地址：your_server_ip
- 用户名：root
- 密码：your_password
- 上传目录：/var/www/html/

### 第五步：配置Nginx

#### 1. 创建配置文件
```bash
vi /etc/nginx/conf.d/dzcyjsyjy.conf
```

#### 2. 添加以下配置
```nginx
server {
    listen 80;
    server_name www.dzcyjsyjy.com dzcyjsyjy.com;
    
    root /var/www/html;
    index index.html index.htm;
    
    location / {
        try_files $uri $uri/ =404;
    }
    
    # 开启gzip压缩
    gzip on;
    gzip_types text/plain text/css application/json application/javascript text/xml application/xml;
    gzip_min_length 1000;
    
    # 缓存静态资源
    location ~* \.(jpg|jpeg|png|gif|ico|css|js)$ {
        expires 30d;
        add_header Cache-Control "public, immutable";
    }
}
```

#### 3. 重启Nginx
```bash
nginx -t  # 测试配置
systemctl restart nginx
```

### 第六步：配置SSL证书（HTTPS）

#### 1. 安装Certbot
```bash
# CentOS
yum install -y certbot python3-certbot-nginx

# Ubuntu
apt install -y certbot python3-certbot-nginx
```

#### 2. 申请免费SSL证书
```bash
certbot --nginx -d www.dzcyjsyjy.com -d dzcyjsyjy.com
```

#### 3. 自动续期
```bash
crontab -e
# 添加以下内容
0 0 1 * * /usr/bin/certbot renew --quiet
```

### 第七步：域名解析配置

#### 1. 登录阿里云域名控制台
#### 2. 添加解析记录：

| 记录类型 | 主机记录 | 记录值 | TTL |
|---------|---------|--------|-----|
| A | @ | 服务器IP | 600 |
| A | www | 服务器IP | 600 |
| CNAME | * | @ | 600 |

## 🔒 安全配置

### 1. 防火墙配置
```bash
# 开放必要端口
firewall-cmd --permanent --add-service=http
firewall-cmd --permanent --add-service=https
firewall-cmd --reload
```

### 2. 修改SSH端口
```bash
vi /etc/ssh/sshd_config
# 修改 Port 22 为其他端口
systemctl restart sshd
```

### 3. 禁用root远程登录
```bash
# 创建新用户
useradd admin
passwd admin
usermod -aG wheel admin

# 修改SSH配置
vi /etc/ssh/sshd_config
PermitRootLogin no
```
